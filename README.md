### Euler Totient - phi(n), Φ(n)
  Number of co-primes(GCD==1) between n and i (1 to n-1)
  - brute force solution (finging all co-primes) -> nlogn  
  - 
  Example : phi(5)= 4  
  1- if n is prime -> phi(n) = n-1  
  2- if a = p^n -> phi(a)= p^n - p^(n-1)
